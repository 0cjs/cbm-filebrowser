Use the proprer program according with your machine.


The file "fb" is a simple basic program that detects the machine
and uses the proper program for the system detected.
It is useful if you use your sd2iec firmware based drive with different systems,
If you put it as first file of the root/folder you need just a LOAD"*",8 from
any system.

If you use just one machine, you may unpack only the proper program for your 
system, put it also as first file for a quick load.
